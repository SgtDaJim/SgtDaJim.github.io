Linux学习笔记 - Linux下常用命令

Posted by Ro$es in Technology

Tags Linux  CentOS  shell 

参考资料：《循序渐进Linux第二版》

系统管理与维护

ls 显示指定工作目录下的内容，列出工作目录所含的文件以及子目录。

pwd 显示当前工作目录。

cd 改变当前工作目录。

date 显示或修改系统日期时间。但只有超级用户才能设置时间。

passwd 设置用户密码。

su 改变用户身份。
注：su命令中使用-参数可以加载相应用户下的环境变量。 如直接输入su -，则会改为root身份，并且加载root的环境变量。 若不加载root的环境变量，某些指令可能无法执行，会提示"command not found"

clear 清屏。

man 显示指定指令的帮助信息。

who 显示目前登录到系统的用户。

w 显示登录到系统的用户信息。

  • 第一行显示当前系统时间、系统运行时间、登录用户数和系统平均负载（1min、5min、15min内系统负载情况）。
  • USER 登录系统的用户
  • TTY 用户使用的TTY
  • FROM 用户登录的远程主机的IP或主机名
  • LOGIN@ 用户登录的日期时间
  • IDLE 某个程序执行持续时间
  • JCPU 终端上所有进程和紫禁城呢个使用系统的总时间
  • PCPU 活动进程使用的系统时间
  • WHAT 用户执行的进程名称和选项

uname 显示操作系统的相关信息。

uptime 输出系统任务队列信息。

显示当前系统时间、系统运行时间、登录用户数和系统平均负载（1min、5min、15min内系统负载情况）。

last 目前与过去登入系统的用户相关信息。（默认读取/var/log/wtmp，并把该文件记录的登入系统的用户名单全部显示出来）

dmesg 显示开机信息。（保存在/var/log/dmesg）

free 显示系统内存状态

ps 显示系统进程在瞬间的运行状态

top 对系统处理器状态的实时监控，能够实时显示系统中各个进程的资源占用情况。

文件管理与编辑

mkdir 创建目录

more 分屏显示较长文本文件。按enter显示文件下一行，按空格显示文件下一屏。

cat 将文件内容输出到标准输出。还可以用于合并文件。

diff 逐行比较文本文件的差异；比较目录中相同文件名的文件。

grep 根据自定字符串，搜索文件每一行。输出字符串所在行内容。

rm 删除某个目录及其下的所有文件和子目录。（若为链接文件，则断开链接，源文件不变。）

touch 改变指定文件的访问时间和修改时间。若文件不存在，则创建文件。

ln 在文件或目录之间创建链接。（默认硬链接）

file 显示文件类型。

cp 将一个文件或目录复制到另一个文件或目录中。

find 指定路径下查找指定文件。

split 分割文档。

mv 将文件和目录重命名或将文件从一个目录移动到另一个目录。

压缩与解压

zip/unzip 压缩、解压。默认压缩文件后缀为".zip"。

gzip/gunzip 压缩、解压。默认压缩文件后缀为“.gz”

bzip2/bunzip 压缩、解压。默认压缩文件后缀为“.bz2”

tar 归档、打包工具。（不压缩）（可以加参数进行压缩）

dd 转换或复制文件，同时可以对设备进行备份。
注：of不能与if在同一个目录下，否则陷入死循环 注：恢复目标分区时，一定要先卸载目标分区，恢复完成后再进行挂载。 注：备份的分区和恢复的分区最好一致，恢复分区若大于备份分区，则浪费空间；否则提示空间不足。

cpio 通过重定向的方式将文件进行打包、备份、还原、恢复的工具。可以解压.cpio或者.tar结尾的文件。
注：cpio在打包、备份时用的是绝对路径，则恢复时会自动恢复到该绝对路径下。若使用相对路径，则会还原到相对路径。

磁盘管理与维护

df 检查Linux系统的磁盘空间占用情况。

du 显示文件或目录所占用的磁盘空间情况。

fsck 检查文件系统并尝试修复错误。
注：fsck修复某个文件系统时，这个文件系统对应的磁盘分区一定要处于卸载状态，磁盘分区在挂载状态下进行修复是极为不安全的，数据可能遭到破坏，也有可能损坏磁盘。

sync 强制把内存中的数据写回硬盘，以免数据的丢失。

eject 退出抽取式设备，如光驱或磁盘。若设备已挂载，eject将先卸载设备，再退出设备。

mount/umount 挂载/卸载指定文件系统

网络设置与维护

ifconfig 配置网络或显示当前网络接口状态

**注：ifconfig配置的网卡信息在网卡或者机器重启后，所有配置失效，要配置永久生效，要修改网卡配置文件。

scp secure copy，将文件或目录从一个Linux系统复制到另一个Linux系统。
scp传输数据使用了SSH协议，保证了数据传输的安全。

netstat 显示本机网络链接、运行端口和路由表等信息。

traceroute 显示网络数据包传输到指定主机的路径信息，追踪数据传输路由状况。（使用了ICMP协议）

telnet 通过telnet协议与远程主机通信或获取远程主机对应端口的信息。

wget 从网络上下载某个文件。
